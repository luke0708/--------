# 📖 USER_MANUAL (说明书)

本手册面向项目管理员，说明如何监控和操作已部署的服务。

## 1. 管理中心 (Infra-Dash)
- **访问地址**: `http://YOUR_VPS_IP:39999`
- **功能**:
    - 查看当前系统所有运行中的容器。
    - 一键触发服务器端的代码更新与重启。
    - 简单的系统资源监控。

## 2. 日志监控 (Dozzle)
- **访问地址**: `http://YOUR_VPS_IP:39998`
- **功能**:
    - **实时日志**: 无需 SSH 到服务器即可查看所有容器的运行日志。
    - **多容器对比**: 支持搜索、过滤和下载日志文件。

## 3. 业务服务 (以 News-Tracker 为例)
- **访问地址**: `http://YOUR_VPS_IP:50003`
- **操作**:
    - 访问前端页面查看聚合的新闻内容。
    - 后台自动根据 `.env` 中配置的 API Key 进行内容处理。

## 4. 常见问题 (FAQ)
- **无法访问管理页面?**
    - 检查 VPS 防火墙（Security List）是否开放了 `39998`, `39999` 端口。
- **数据没有更新?**
    - 使用 Dozzle 查看 `news-tracker` 容器日志，检查是否 API Key 额度用光或过期。
