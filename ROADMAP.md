# 🗺️ 项目蓝图与演进规划 (Roadmap)

本项目致力于成为最轻量且全自动的私有云部署方案。

---

## 🚩 已完成 (V1.0 - 自动化雏形)
- [x] 基于 AI 协议的 Docker 自动化部署逻辑。
- [x] 全局端口管理注册表 (Port-Registry)。
- [x] 基础监控面板 (Infra-Dash + Dozzle)。
- [x] 第一个核心业务应用：AI 新闻追踪系统 (News-Tracker)。

## 🚧 进行中 (V1.1 - 智能化增强)
- [ ] **主题智能填充**：根据中文名自动补全英文名与关键词（AI 驱动）。
- [ ] **摘要深度化**：优化 LLM Prompt，增加新闻对特定金融资产（黄金/汇率）的影响分析。
- [ ] **稳定性加固**：增加自动健康检查重试机制，若容器失败自动回滚。

## 📅 未来展望 (V2.0 - 云端协同)
- [ ] **多端同步控制**：通过 Telegram/飞书机器人实时推送部署状态并进行简单控制。
- [ ] **静态资源 CDN 自动部署**：将前端打包文件自动同步至 OSS/CDN。
- [ ] **智能算力分配**：根据 Mac Air 负载自动调整抓取频率。
- [ ] **一键迁移**：支持通过一条指令将整个 Infrastructure 迁移至另一台宿主机。
