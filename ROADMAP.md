# 🛣️ ROADMAP.md (项目规划)

本计划列出了 Infrastructure 及相关业务服务的演进路线。

## 📅 短期目标 (Q1 2026)
- [ ] **反向代理集成**: 引入 Nginx Proxy Manager，实现 HTTPS 访问。
- [ ] **内存优化**: 针对 1GB VPS，研究使用更轻量级的镜像（如编译后运行）。
- [ ] **备份机制**: 自动定期备份 `news.db` 等关键数据库文件到 S3 或云存储。

## 📅 中期目标 (Q2 2026)
- [ ] **Infra-Dash 增强**:
    - 支持在网页端编辑 `.env` 文件（需要权限隔离）。
    - 集成更多的系统指标图表（CPU/内存实时预览）。
- [ ] **自动缩减**: 长期不活跃的服务自动转为睡眠状态，释放 VPS 内存。

## 📅 长期目标 (2027+)
- [ ] **多机容灾**: 支持一键将 Infrastructure 环境复制到另一台 VPS，实现双机高可用。
- [ ] **自研网关**: 替换掉 Nginx，使用更贴合本项目架构的轻量级 Golang 网关。
